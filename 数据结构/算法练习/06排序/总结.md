## 1.1 插入排序

| 排序类别 | 排序算法     | 时间复杂度 |        |          | 辅助空间复杂度 | 稳定性 |
| -------- | ------------ | ---------- | ------ | -------- | -------------- | ------ |
|          |              | 最好       | 最坏   | 平均     |                |        |
| 插入排序 | 直接插入排序 | O(n)       | O(n^2) | O(n^2)   | O(1)           | 稳定   |
| 插入排序 | 折半插入排序 | O(n)       | O(n^2) | O(n^2)   | O(1)           | 稳定   |
| 插入排序 | 希尔排序     | O(n)       | O(n^2) | O(n^1.3) | O(1)           | 不稳定 |
